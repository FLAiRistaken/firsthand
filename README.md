# Firsthand

A personal mobile app for building awareness of AI reliance and rebuilding your own thinking capabilities.

## Concept
Log wins (I did it myself) and sins (I used AI). Build awareness over time. The goal is intentional AI use, not zero use.

## Stack
- React Native (Expo)
- Supabase (auth + database + sync)
- Anthropic API (onboarding + Socratic coach)
